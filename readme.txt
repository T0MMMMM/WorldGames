Salut, quelques information pour lancer le jeu :

1- Ouvrir le dossier WorldGames avec VScode
2- Executez la commande "/cd src" puis "/go run ." pour lancer le jeu !


information du jeu:
    Au lancement, vous avez le choix entre deux jeux, le wordle et le hangman:
    
    -pour le hangman, il suffit de cliquer sur les lettres du clavier. Vous pouvez appuyer plusieurs fois sur la même lettre,
    cela ne comptera pas.

    -pour le wordle, il faut écrire un mot avec le clavier et le valider avec la touche entré,  si cela ne marche pas,
    c'est que le mot n'est pas dans le répertoire du jeu.

    Pour avoir un nouveau mot, il faut relancer le programme.
